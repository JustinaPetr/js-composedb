# Data Interactions
Query and mutate data.

## Overview

After setting up your [ComposeDB Client](../guides/composedb-client/javascript-client), you can perform queries and mutations on ComposeDB data.

## Getting Started

### Queries

[**Queries**](./data-interactions/queries) allow you to fetch data.

### Mutations

[**Mutations**](./data-interactions/mutations) allow you to create or update data.
